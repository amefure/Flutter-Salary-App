# シンプル給料記録アプリ

給料や副業収益を記録・管理するためのメモアプリ

## アプリ概要

＼シンプルな給料・副業収入記録アプリ／

◇このアプリでできること

1. 給料を記録
2. 年収や手取りが把握できる
3. グラフで増減が見える化
4. 副業やバイトなどの収益も別で管理可能
5. アプリにロックもかけられる(生体認証(指紋/顔)でログイン)

追加して欲しい機能や改善してほしい箇所がありましたら気軽にレビューから教えてください！
